1. What is DOM and what is the purpose of DOM?

Answer: DOM is the short name for the Document Object Model. It is an application programming interface (API) for HTML and XML documents. DOM is used for the dynamic usage of the scripts. It helps to update the document's content, structure, and style. DOM is also used to create objects.

3. What is event bubble?

Answer: Event Bubbling is a way to reach out to parents of an event by propagating. When we call an element then it will find all its children by propagating. It will bubble up until gets to the root element.

4. What is a callback function and why will you use it?

Answer: When we need to run a function after doing something then we need to use callback function. The result of the completed task usually uses as an argument of the callback function.
